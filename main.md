## 本機儲存規劃 (我的最愛)
為了保持 App 最簡架構、完全不依賴後端資料庫，使用者的「我的最愛 (Favorites)」將儲存在手機/瀏覽器本機端：
* **核心機制**：採用 Web 標準 `localStorage` 進行儲存，儲存鍵值為 `vege_favorites`。
* **資料結構**：以 JSON 格式儲存包含最愛作物的代碼與名稱陣列，例如：
  ```json
  ["FR715", "00208075001999950"]
  ```
* **運作邏輯**：
  1. **載入**：App 啟動時讀取 `localStorage.getItem('vege_favorites')`，若無則預設為 `[]`。
  2. **寫入**：當使用者點擊收藏/取消收藏時，更新 React 狀態，同步執行 `localStorage.setItem('vege_favorites', JSON.stringify(newFavs))`。
  3. **篩選**：點擊「我的最愛」分類標籤時，`filterCrops()` 將依據該最愛代碼/名稱陣列進行資料過濾與呈現。
